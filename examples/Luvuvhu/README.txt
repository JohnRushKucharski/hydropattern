This README explains the data/ directory's contents:

This directory contains observed and modeled hydrologic flows for the Luvuvuhu river used in the UNESCO SA Biospheres project.

Directory:

data/
|- wflow/
|  |-
|- wrc/
|  |- all-in-one.zip
|- A9H012.txt
|- readme.txt

wflow/
- cst_summary.csv - explains wflow hydrologic flow simulation data climate scenario naming scheme (run from cst)
- no_action.csv - baseline and future climate scenario simulation data without adaptive actions
- pathways-historical.zip - contains simulations with adaptive actions.
wrc/
- all-in-one.zip - monthly naturalized flow data in millions of cubic meters (1920-2009) taken from: 
	https://waterresourceswr2012.co.za/ 
	email: john.kucharski@deltares.nl password: John 
	> 9. Naturalised flow datafiles
A9H012.txt - observed streamflow near the outlet of the wflow modeled basin.
